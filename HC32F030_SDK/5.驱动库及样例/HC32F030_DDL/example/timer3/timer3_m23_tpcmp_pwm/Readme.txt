功能描述
================================================================================
本样例展示Base Timer3的CHA与CHB输出独立PWM的双点比较功能，
使用CCRxA和CCRxB控制OCREFA（CHxA）的比较输出功能（双点比较功能，非对称互补PWM）,
CCRxB控制OCREFB（CHxB）的比较输出功能

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
双通道示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器探头连接到PA08(TIM3 CH0A)和PA07(TIM3 CH0B)，PB10(TIM3 CH1A)和PB00(TIM3 CH1B)，
PB08(TIM3 CH2A)和PB15(TIM3 CH2B)
4）运行，观察波形输出
5）设置间隔计数为1，在UEV更新中断内对CCRxA和CCRxB的进行更改，CHxA与CHxB的占空比相隔
一个周期更替变化。
可以看到，CHxA的上升沿时机由CCRxA设定的值控制，CHxA的下降沿时机由CCRxB设定的控制。

================================================================================
注意
================================================================================
1）该样例展示的是CHxA与CHxB输出独立PWM的双点比较功能（CHxA的左右边沿分别CCRxA和CCRxB设置）

2）如果要测试CHxA与CHxB的互补PWM功能（双点比较：CHxA占空比由CCRxA和CCRxB共同设置，CHxB与CHxA互补），
请将stcTim3BaseCfg.enPWMTypeSel  = Tim3IndependentPWM; ==》改为 stcTim3BaseCfg.enPWMTypeSel  = Tim3ComplementaryPWM;

3）如果要测试CHA与CHB的互补PWM功能（单点比较：CHA占空比由CCR0A设置，CHB与CHA互补）
请将stcTim3BaseCfg.enPWMTypeSel  = Tim3IndependentPWM; ==》改为  stcTim3BaseCfg.enPWMTypeSel  = Tim3ComplementaryPWM;
stcTim3BaseCfg.enPWM2sSel    = Tim3DoublePointCmp;     ==》改为  stcTim3BaseCfg.enPWM2sSel    = Tim3SinglePointCmp;
