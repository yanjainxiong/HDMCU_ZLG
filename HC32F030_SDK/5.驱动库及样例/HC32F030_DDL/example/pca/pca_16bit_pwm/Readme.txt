功能描述
================================================================================
本样例主要展示PCA的PWM功能。
说明：
本样例展示PCA的16位PWM功能，通过设定不同的数值，产生不同占空比的波形。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）连接示波器和PA06（PCA_CH0)
4）运行，观察波形


================================================================================
注意
================================================================================
1）用户在实际应用中可以通过更改比较捕获寄存器CCAPx的值来更改PWM占空比，
   通过更改周期重载值CARR来更改PWM周期
