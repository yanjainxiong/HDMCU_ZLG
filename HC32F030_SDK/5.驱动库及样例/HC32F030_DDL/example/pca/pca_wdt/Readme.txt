功能描述
================================================================================
本样例主要展示PCA的WDT功能。
说明：
本样例展示PCA的WDT功能，在达到定时值时如果没有及时设定比较值（喂狗），则发生系统复位；
反之则系统正常运行。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1) 通过宏定义选择喂狗或不喂狗功能
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）在"main()"第一行设置断点；
5）运行，到断点处停止，继续运行
6）若（1）选择的是喂狗功能，则程序继续运行，不产生复位
7）若（1）选择的是不喂狗功能，则程序复位，停止在断点处。
