功能描述
================================================================================
本样例主要展示MCU的VC的输出到通用定时器

说明：
开启VC上升沿中断请求
调整P端口电压，板上LED灯变化

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  连接P端PC00接外部直流稳压源。
2）  打开工程编译并运行。
3）  调整直流电源到1.5V以上。
4）  观察板上LED灯变化
5）  调整直流电源到1.5V以下，观察板上LED灯变化。
6）  重复4-6。板上LED等会在电量和熄灭状态切换
