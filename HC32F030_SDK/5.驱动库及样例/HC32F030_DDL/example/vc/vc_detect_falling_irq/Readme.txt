功能描述
================================================================================
本样例主要展示MCU的VC的下降沿中断功能

说明：
开启VC下降沿中断请求
调整N/P端口电压，观察VC0OUT PA06端口电平变化

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  连接PA00到VCORE1.5v端口，P端PC00接外部直流稳压源。
2）  打开工程编译并运行。
3）  调整直流电源到1.5V以上。
4）  测试脚PA06输出电平翻转。
5）  调整直流电源到1.5V以下，测试脚输出电平没有变化,观察PD05端口电平变化与触发方式的关系。
6）  重复4-6。
