功能描述
================================================================================
本样例展示Base Timer的带硬件死区互补PWM输出功能，并触发ADC的插队扫描采样。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
双通道示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器探头连接到PA00(TIM0 CHA)和PA01(TIM0 CHB)
4）运行，观察波形输出（两路互补PWM，并且带有死区时间）;
5）设置间隔计数为1，每个周期Uev触发中断，并触发ADC采样。
