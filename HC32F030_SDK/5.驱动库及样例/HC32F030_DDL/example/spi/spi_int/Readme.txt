---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译
2）连接GT25C64芯片
3）运行程序，在live watch（IAR）或者watch(Keil)窗口观察Buff[]数组，可以观察到
   buff[3]~buff[7]的数据与buff[13]~b[17]的数据相同，说明发送与接收一致。
4) 数据若接收相同，PA00电平置高，否则为低。

================================================================================
注意
================================================================================
1）SPI 相应的引脚设置
   PA15：CS
   PA11：MISO
   PA12：MOSI
   PB03：SCK
