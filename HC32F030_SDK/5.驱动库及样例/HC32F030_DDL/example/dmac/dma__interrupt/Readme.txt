功能描述
================================================================================
本样例主要展示ADC 扫描连续采样累加示例，使用中断方式获取ADC运行状态和ADC的采样值。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）在DMA中断内比较传输值，并设置转换完成Flag。
