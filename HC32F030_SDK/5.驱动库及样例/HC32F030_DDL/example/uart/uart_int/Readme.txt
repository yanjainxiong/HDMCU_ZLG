功能描述
================================================================================
本样例主要涉及串口中断收发数据样例

说明：
配置PA02、PA03为UART收发端口
配置波特率9600bps
使能UART中断
使能UART收发功能
================================================================================
测试环境
================================================================================
测试用板:
HC32F030_STK
---------------------
（需要填）

辅助工具:
---------------------
串口线

辅助软件:
---------------------
串口调试工具

================================================================================
使用步骤
1、重新编译代码运行
2、串口调试工具配置波特率9600bps、偶校验、1stop
3、PC串口发送1字节数据XX
4、发送完成后，PC机接收数据XX+0x55
