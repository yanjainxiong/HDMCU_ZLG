功能描述
================================================================================
本样例主要展示CRC32编码及校验功能。
说明：
本样例先对测试数据进行CRC32编码，然后使用校验功能对编码结果进行校验，
样例中的测试数据来自CRC32编解码软件。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
CRC32编解码软件。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）在main函数while(1)处设置断点；
4）运行;
5）程序到断点处停止，此时观察"enResult"的值，若为'0'则编码及校验功能正常。

================================================================================
注意
================================================================================
CRC16的编码测试数据及CRC均通过CRC32编解码软件计算获得，程序中会将实际计算结果与编码软件计算结果进行比对。
