功能描述
================================================================================
本样例主要展示ADC 被TIM4触发启动顺序扫描采样，TIM4的触发频率是50K


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）Timer4每隔20us触发ADC采样
3）在ADC中断内读取转换值
