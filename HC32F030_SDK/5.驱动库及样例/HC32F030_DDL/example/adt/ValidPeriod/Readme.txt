功能描述
================================================================================
本样例主要展示ADT周期间隔响应示例，通过配置有效周期间隔，触发ADC采样。

================================================================================

---------------------
HC32F030_STK

示波器

---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）间隔配置的有效周期，每隔1个周期
3）下溢匹配产生中断并触发ADC采样
