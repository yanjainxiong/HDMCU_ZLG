测试环境
================================================================================

---------------------
HC32F030_STK

---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）各ADT定时器软件同步开始、同步停止、同步清零功能正常
3）在相应的停止和清零处设置断点，可以看到‘stcAdt4CntState.u16Counter’，
‘stcAdt5CntState.u16Counter’和‘stcAdt6CntState.u16Counter’数值同步。
