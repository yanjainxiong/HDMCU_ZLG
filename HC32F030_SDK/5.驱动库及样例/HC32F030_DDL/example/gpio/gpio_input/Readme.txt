功能描述
================================================================================
本样例主要展示GPIO输入功能。
说明：
本样例使用PD05（USER KEY）作为输入功能演示，并根据输入的值设置LED的状态。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F030_STK

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）将程序下载到Demo板上；
4）重新上电运行;
5）按下USER KEY，LED点亮；释放USER KEY，LED关闭。
